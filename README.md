# Loto6 Winning Number Scraper

## 概要
みずほ銀行の公式サイトから、ロト6の過去の当選番号を自動抽出し、データ分析に活用しやすいCSV形式で保存するスクレイピングツールです。

## 作成背景
「過去の当選番号の傾向を分析したい」という課題に対し、手動でデータを収集するのは非効率であると考え、自動化プログラムを作成しました。
また、スクレイピングの実践を通じて、動的サイトの制御（Selenium）と静的なパース（BeautifulSoup）の組み合わせ、およびWebエチケットの遵守について学ぶことを目的としています。

## 主な機能
- **自動ページネーション**: 過去から最新回までの複数ページにまたがるデータを自動巡回。
- **データクレンジング**: 取得した生データを整形し、本数字とボーナス数字を分離して構造化。
- **CSV出力**: Pandasを用いて、分析に適した形式でローカルへ保存。

## こだわった点・工夫した点
- **Webサイトへの負荷軽減**: 
    - `time.sleep`と`random.uniform`を組み合わせ、不規則な待機時間を設けることでサーバーへの負荷を最小限に抑えています。
- **実行速度の最適化**: 
    - ChromeのHeadlessモードを使用し、画像の読み込みを無効化することで、リソース消費を抑えた高速な実行を実現しました。
- **保守性の考慮**: 
    - 将来的なURL構造の変化に対応しやすいよう、回数に応じたURL生成ロジックを条件分岐で管理しています。

## 使用技術
- **Language**: Python 3.x
- **Libraries**:
  - `Selenium`: ブラウザ操作・動的コンテンツの取得
  - `BeautifulSoup4`: HTMLパース
  - `Pandas`: データ整形・CSV出力
  - `webdriver-manager`: ブラウザドライバの自動管理
